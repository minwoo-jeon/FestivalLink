## 프로젝트 소개 
- 사용자 맞춤형 축제정보를 인기순/최신순으로 검색할수 있습니다
- 축제 커뮤니티에서 사용자들과 축제 후기를 공유할수 있습니다.
- 관심있는 축제들을 좋아요&스크랩 기능을 통해 축제 피드들을 모아 관리 할수 있습니다.
- 사용자가 직접 소행사 게시판을 통해 행사 홍보를 할수있습니다.
